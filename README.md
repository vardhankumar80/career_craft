# Career Craft

**Career Craft** is a full‑stack web application that helps individuals explore, plan and navigate their career paths.  
It features a user‑friendly front‑end, a robust back‑end API, and tools to manage career‑related resources and workflows.

## Features  
- User registration and authentication  
- Profile management and skill‑tracking  
- Explore career paths, job roles, and skill‑gaps  
- Interactive dashboard with progress‑tracking  
- Responsive UI for desktop & mobile  
- Extensible architecture for adding new modules



## Tech Stack  

**Frontend**: React, CSS/Sass, Responsive design  
**Backend**: Node.js + Express , RESTful API  
**Database**: MongoDB / PostgreSQL  
**Other Tools**: JWT authentication, bcrypt for hashing, etc.

## Getting Started  

### Prerequisites  
- Node.js (v14 or higher)  
- npm or yarn  
- MongoDB / PostgreSQL instance

### Usage

- Register a new account or log in with existing credentials.
- Create and update your profile, including skills and interests.
- Explore suggested career paths and recommended skill improvements.
- Track your career development progress through the dashboard.

### Installation  
```bash

npm install

cd ../frontend  
npm install  

# In one terminal window:

npm run dev  

# In another terminal:  
cd frontend  
npm start
