# Refined Menu Page

A fully responsive and animated menu page built with HTML, CSS, and JavaScript.  
This project demonstrates a clean design, motion effects, and dropdown navigation while keeping a consistent theme.

## Features

- Responsive layout for mobile, tablet, and desktop
- Smooth hover and motion animations for menu items and cards
- Dropdown menu for services with transition effects
- Modern design with a consistent color theme
- Easy to customize and extend

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Transitions)
- JavaScript (Vanilla JS)
